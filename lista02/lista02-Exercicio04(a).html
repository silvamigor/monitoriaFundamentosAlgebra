Lista 2

Seja \(a\) um número natural escrito como \([a_na_{n-1}\dots a_1a_0]_{b}\) na base \(b\), onde \(b \ge 2\). Mostre que

\(b-1 \mid a\) se, e somente se, \(b-1 \mid a_n + a_{n-1} + \cdots + a_1 + a_0\)

Vamos mostrar que para todo \(n \in \bbN\), existe \(q_n \in \bbN\), tal que \(b^n = (b - 1)q_n + 1\). Tal prova será feita por indução em \(n\). Se \(n = 1\), então \(b^1 = (b - 1)1 + 1\). Suponha que vale a igualdade valha para \(k > 1\). Então \[ \begin{split} b^{k+1} &= b^kb = \big((b - 1)q_k + 1\big)(b - 1 + 1) = \\ &= (b - 1)\underbrace{\big((b - 1)q_k + q_k + 1 \big)}_{q_{k+1}} + 1 \end{split} \] Logo, existe \(q_{k+1} = \big((b - 1)q_k + q_k + 1 \big)\) tal que \(b^{k+1} = (b - 1)q_{k+1} + 1\). Assim, se \(a = [a_na_{n-1}\dots a_1a_0]_{b}\), então \begin{align} a &= a_nb^n + \cdots + a_1b + a_0 \nonumber \\ &= a_n((b - 1)q_n + 1) + \cdots + a_1(b - 1 + 1) + a_0 \nonumber \\ &= (b - 1)(a_nq_n + \cdots + a_1) + a_n + \cdots + a_1 + a_0 \label{eq1} \end{align} Dessa forma, se \(b - 1 \mid a\), então existe \(s \in \bbN\) tal que \(a = (b - 1)s\). Substituindo na equação \eqref{eq1}, temos que \[ a_n + \cdots + a_1 + a_0 = (b - 1)(s - a_nq_n - \cdots - a_1), \] ou seja, \(b-1 \mid a_n + a_{n-1} + \cdots + a_1 + a_0\). Reciprocamente, se \(b - 1 \mid \sum_{i=0}^na_i\), então existe \(r \in \bbN\) tal que \(\sum_{i=0}^na_i = (b - 1)r\). Logo, pela equação \eqref{eq1}, temos que \[ a = (b - 1)(a_nq_n + \cdots + a_1 + r), \] isto é, \(b - 1 \mid a\).
