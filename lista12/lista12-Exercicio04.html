Lista 12

Seja \(\bbF\) um corpo. Defina o derivado simbólico de um polinômio

\[ f(x) = \alpha_nx^n + \alpha_{n-1}x^{n-1} + \cdots + \alpha_1x + \alpha_0 \in \bbF[x] \]
como
\[ f'(x) = n\alpha_nx^{n-1} + (n-1)\alpha_{n-1}x^{n-2} + \cdots + \alpha_1x + \alpha_1. \]
(Note que o derivado simbólico não é definido usando limite, e portanto faz sentido para polinômios sobre \(\bbZ_p\)). Um escalar \(\alpha \in \bbF\) é dito raiz múltipla de \(f(x) \in \bbF[x]\setminus\zero\), se \((x - \alpha)^2 \mid f(x)\). Mostre que \(\alpha \in \bbF\) é uma raiz múltipla de \(f(x)\) se, e somente se, \((x - \alpha) \mid \mdc(f(x),f'(x))\).

Vamos mostrar que a derivada formal de polinômios satisfaz a regra do produto, isto é, que \((f(x)g(x))' = f'(x)g(x) + f(x)g'(x)\), para todo \(f(x), g(x) \in \bbF[x]\). Escrevendo \(f(x) = \sum_{i = 1}^{n}\alpha_{i}x^{i}\) e \(g(x) = \sum_{j = 1}^{m}\beta_{j}x^j\), pode-se mostrar que \((f(x) + g(x))' = f'(x) + g'(x)\) e que \((\lambda f(x))' = \lambda f'(x)\), para todo \(\lambda \in \bbF\). Assim, se \(f_1(x)\), \(f_2(x)\), \(g_1(x)\) e \(g_2(x)\) são polinômios em \(\bbF[x]\) tais que satisfazem a regra do produto, isto é, \((f_i(x)g_j(x))' = f_i'(x)g_j(x) + f_i(x)g_j'(x)\), para todo \(i,j = 1, 2\), então

\[ \begin{split} \Big(\big(f_1(x) + f_2(x)\big)\big(g_1(x) + g_2(x)\big)\Big)' &= \big(f_1(x)g_1(x) + f_1(x)g_2(x) + f_2(x)g_1(x) + f_2(x)g_2(x)\big)' \\ &= (f_1(x)g_1(x))' + (f_1(x)g_2(x))' + (f_2(x)g_1(x))' + (f_2(x)g_2(x))' \\ &= f_1'(x)g_1(x) + f_1(x)g_1'(x) + f_1'(x)g_2(x) + f_1(x)g_2'(x) + f_2'(x)g_1(x) + f_2(x)g_1'(x) + f_2'(x)g_2(x) + f_2(x)g_2'(x) \\ &= f_1'(g_1(x) + g_2(x)) + (f_1(x) + f_2(x))g_1'(x) + f_2'(x)(g_1(x) + g_2(x)) + (f_1(x) + f_2(x))g_2'(x) \\ &= \big(f_1'(x) + f_2'(x)\big)\big(g_1(x) + g_2(x)\big) + \big(f_1(x) + f_2(x)\big)\big(g_1'(x) + g_2'(x)\big) \\ &= \big(f_1(x) + f_2(x)\big)'\big(g_1(x) + g_2(x)\big) + \big(f_1(x) + f_2(x)\big)\big(g_1(x) + g_2(x)\big)', \end{split} \]
a soma dos polinômios também satisfazem a regra do produto. Uma vez que polinômios são somas de monômios, então se mostrarmos que \(x^n\) e \(x^m\) satisfazem a regra do produto, concluímos que tal regra vale para quaisquer polinômios. Assim, veja que
\[ \begin{split} (x^nx^m)' &= (x^{n+m})' = (n+m)x^{n+m-1} = nx^{n-1}x^m + mx^{n}x^{m-1} \\ &= (x^n)'x^m + x^n(x^m)'. \end{split} \]
Portanto, para todo \(f(x), g(x) \in \bbF[x]\), vale \((f(x)g(x))' = f'(x)g(x) + f(x)g'(x)\). De posse desse resultado, vamos resolver a questão. Suponha que \(\alpha\) é uma raiz múltipla de \(f(x)\). Então existe \(q(x)\) tal que \(f(x) = (x-\alpha)^2q(x)\). Assim, pela regra do produto,
\[ \begin{split} f'(x) &= ((x-\alpha)^2)'q(x) + (x-\alpha)^2q'(x) = 2(x-\alpha)q(x) + (x-\alpha)^2q'(x) \\ &= (x-\alpha)(2q(x) + (x-\alpha)q'(x)), \end{split} \]
ou seja, \((x -\alpha)\) divide ambos \(f(x)\) e \(f'(x)\). Portanto, \((x - \alpha) \mid \mdc(f(x),f'(x))\). Reciprocamente, suponha que \(\alpha\) não é uma raiz múltipla de \(f(x)\). Então \(f(x) = (x - \alpha)g(x)\), onde \((x - \alpha) \not\mid g(x)\). Assim, pela regra do produto, \(f'(x) = g(x) + (x - \alpha)g'(x)\). Se \((x - \alpha) \mid f'(x)\), então \((x - \alpha) \mid g(x)\) o que é um absurdo. Logo, \((x - \alpha) \not\mid f'(x)\) e, consequentemente, \((x - \alpha) \not\mid \mdc(f(x),f'(x))\).
