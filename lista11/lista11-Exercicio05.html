Lista 11

Sejam \(R\) um anel e \(a \in R\). O elemento \(a\) é dito nilpotente, se existe \(k \ge 1\) tal que \(a^k = 0\). Demonstre as seguintes afirmações.

  1. Se \(R\) é um domínio, então o único elemento nilpotente é \(0\).
  2. Se \(a, b \in R\) são nilpotentes, então \(a + b\) também é nilpotente e \(ac\) é nilpotente, para todo \(c \in R\).
  3. Se \(a \in R\) é nilpotente e \(b \in R\) é invertível, então \(a + b\) é invertível.
  4. Ache todos os elementos nilpotentes de \(\bbZ_{100}\).

Antes de começarmos, lembre-se de que um anel, para essa disciplina, é comutativo com identidade, conforme visto na aula "Anéis, domínios e corpos".

(a) Seja \(a \in R\) um elemento nilpotente. Então \(a^k = 0\), para algum \(k \in \bbN\). Seja \(n \in \bbN\) o menor \(k\) tal que \(a^k = 0\), isto é, \(a^n = 0\) e \(a^i \neq 0\), para todo \(i < n\). Disso, segue que \(aa^{n-1} = 0\). Uma vez que \(R\) é um domínio e \(a^{n-1} \neq 0\), então \(a = 0\).

(b) Sejam \(a, b \in R\) elementos nilpotentes. Então existem \(n,m \in \bbN\) tais que \(a^n = 0\) e \(b^m = 0\). Note que

\[ \begin{split} (a + b)^{n+m} &= \sum_{i = 0}^{n+m}\binom{n+m}{i}a^{i}b^{n+m-i} \\ &= \underbrace{\binom{n+m}{0}b^{n+m} + \binom{n+m}{1}ab^{n+m-1} + \cdots + \binom{n+m}{n}a^{n}b^{m}}_{\text{ é igual a } 0, \text{ pois } b^m = 0} + \underbrace{\binom{n+m}{n+1}a^{n+1}b^{m-1} + \binom{n+m}{n+2}a^{n+2}b^{m-2} + \cdots + \binom{n+m}{n+m}a^{n+m}}_{\text{ é igual a } 0, \text{ pois } a^n = 0} \\ &= 0. \end{split} \]
Logo, \(a+b\) é nilpotente. Agora, seja \(c \in R\). Então \((ac)^n = a^nc^n = 0\). Portanto, \(ac\) também é nilpotente.

(c) Seja \(a \in R\) um elemento nilpotente e \(b \in R\) um elemento invertível. Então existe \(n \in \bbN\) e \(b^{-1} \in R\) tais que \(a^n = 0\) e \(bb^{-1} = 1\). Assim, temos que \(b^{-1}(a + b) = b^{-1}a + 1\). Observe, agora, que

\[ (1 + x)(1 - x + x^2 - x^3 + \cdots + (-1)^{n-1}x^{n-1}) = 1 + (-1)^{n-1}x^{n}, \]
para todo \(x \in R\). Em particular, vale para \(x = b^{-1}a\). Definindo \(f(x) = 1 - x + x^2 - x^3 + \cdots + (-1)^{n-1}x^{n-1}\), temos que \((1 + b^{-1}a)f(b^{-1}a) = 1 + (-1)^{n-1}(b^{-1}a)^{n}\). Uma vez que \(a^n = 0\), então \((1 + b^{-1}a)f(b^{-1}a) = 1\). Como \(b^{-1}(a + b) = b^{-1}a + 1\), então \((a + b)b^{-1}f(b^{-1}a) = 1\), ou seja,
\[ b^{-1}\big(1 - b^{-1}a + (b^{-1}a)^2 - (b^{-1}a)^3 + \cdots + (-1)^{n-1}(b^{-1}a)^{n-1}\big) \]
é o inverso de \(a + b\).

(d) Seja \(\overline{n} \in \bbZ_{100}\) um elemento nilpotente. Então existe \(k \in \bbN\) tal que \(\overline{n}^k = \overline{0}\). Isso significa que \(n^k \equiv 0 \pmod{100}\), ou ainda, que \(100 \mid n^k\). Isso equivale a \(2 \mid n\) e \(5 \mid n\). Portanto, os valores em \(\bbZ_{100}\) que são nilpotentes são: \(0\), \(10\), \(20\), \(30\), \(40\), \(50\), \(60\), \(70\), \(80\) e \(90\).
