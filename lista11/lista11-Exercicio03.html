Lista 11

Seja \(p\) um primo tal que \(p \equiv 3 \pmod{4}\). Assuma que \(i\) é um símbolo tal que \(i^2 = \overline{-1} \in \bbZ_p\). Defina \[ \bbF_{p^2} = \{a + bi \mid a, b \in \bbZ_p\} \] com as operações \[ \begin{array}{rcl} (a_1 + b_1i) + (a_2 + b_2i) & = & (a_1 + a_2) + (b_1 + b_2)i; \\ (a_1 + b_1i)(a_2 + b_2i) & = & (a_1a_2 - b_1b_2) + (a_1b_2 + a_2b_1)i. \end{array} \] Mostre que \(\bbF_{p^2}\) é um corpo. Explique porque a condição \(p \equiv 3 \pmod{4}\) é necessária.

Primeiro observe que as operações adição e multiplicação, como definidas, são operações que atribuem a um par ordenado de \(\bbF_{p^2}\) a um elemento de \(\bbF_{p^2}\). Isso quer dizer essa operações são fechadas, que não saem do conjunto. Portanto, estão bem definidas. Vamos mostrar que elas satisfazem as propriedades que caracterizam corpos.

• Associatividade da adição. \[ \begin{split} \big((a_1 + b_1i) + (a_2 + b_2i)\big) &+ (a_3 + b_3i) = \big((a_1 + a_2) + (b_1 + b_2)i\big) + (a_3 + b_3i) \\ &= \big((a_1 + a_2) + a_3\big) + \big((b_1 + b_2) + b_3\big)i \\ \text{pela associatividade de } \bbZ_p &= \big(a_1 + (a_2 + a_3)\big) + \big(b_1 + (b_2 + b_3)\big)i \\ &= (a_1 + b_1i) + \big((a_2 + a_3) + (b_2 + b_3)i\big) \\ &= (a_1 + b_1i) + \big((a_2 + b_2i) + (a_3 + b_3i)\big). \end{split} \]

• Comutatividade da adição. \[ \begin{split} (a_1 + b_1i) + (a_2 + b_2i) &= (a_1 + a_2) + (b_1 + b_2)i \\ \text{pela comutatividade de } \bbZ_p &= (a_2 + a_1) + (b_2 + b_1)i \\ &= (a_2 + b_2i) + (a_1 + b_1i). \end{split} \]

• Existência do elemento neutro aditivo. Seja \(\overline{0} \in \bbZ_p\) o elemento neutro aditivo, então \(\overline{0} + \overline{0}i\) é o elemento neutro aditivo de \(\bbF_{p^2}\), pois \[ (a + bi) + (\overline{0} + \overline{0}i) = a + bi. \]

• Existência do inverso aditivo. Para cada \(x \in \bbZ_p\), temos que \(-x \in \bbZ_p\) é seu inverso aditivo. Logo, \(-a + (-b)i\) é o inverso aditivo de \(a + bi\) em \(\bbF_{p^2}\), pois \[ (a + bi) + (-a + (-b)i) = \overline{0} + \overline{0}i. \]

• Associatividade da multiplicação. \[ \begin{split} \big((a_1 + b_1i)&(a_2 + b_2i)\big)(a_3 + b_3i) = \big(a_1a_2 - b_1b_2 + (a_1b_2 + a_2b_1)i\big)(a_3 + b_3i) \\ &= (a_1a_2 - b_1b_2)a_3 - (a_1b_2 + a_2b_1)b_3 + \big((a_1a_2 - b_1b_2)b_3 + a_3(a_1b_2 + a_2b_1)\big)i \\ &= (a_1a_2)a_3 - (b_1b_2)a_3 - (a_1b_2)b_3 - (a_2b_1)b_3 + \big((a_1a_2)b_3 - (b_1b_2)b_3 + a_3(a_1b_2) \\ \text{pela associatividade} & \hspace{15cm} + a_3(a_2b_1)\big)i \\ \text{e comutatividade} &= a_1(a_2a_3) - b_1(a_3b_2) - a_1(b_2b_3) - b_1(a_2b_3) + \big(a_1(a_2b_3) - (b_2b_3)b_1 + a_1(a_3b_2) \\ \text{de } \bbZ_p & \hspace{15cm} + (a_2a_3)b_1\big)i \\ &= a_1(a_2a_3 - b_2b_3) - b_1(a_2b_3 + a_3b_2) + \big(a_1(a_2b_3 + a_3b_2) + (a_2a_3 - b_2b_3)b_1\big)i \\ &= (a_1 + b_1i)\big(a_2a_3 - b_2b_3 + (a_2b_3 + a_3b_2)i\big) \\ &= (a_1 + b_1i)\big((a_2 + b_2i)(a_3 + b_3i)\big). \end{split} \]

• Comutatividade da multiplicação. \[ \begin{split} (a_1 + b_1i)(a_2 + b_2i) &= (a_1a_2 - b_1b_2) + (a_1b_2 + a_2b_1)i \\ \text{pelas propriedades de } \bbZ_p &= a_2a_1 - b_2b_1 + (a_2b_1 + a_1b_2)i \\ &= (a_2 + b_2i)(a_1 + b_1i). \end{split} \]

• Existência do elemento neutro multiplicativo. Como \(\overline{1}\) é o elemento neutro multiplicativo em \(\bbZ_p\), então o elemento neutro multiplicativo de \(\bbF_{p^2}\) é \(\overline{1} + \overline{0}i\), pois \[ (a + bi)(\overline{1} + \overline{0}i) = a\overline{1} - b\overline{0} + (a\overline{0} + \overline{1}b)i = a + bi. \]

• Distributividade. \[ \begin{split} (a_1 + b_1i)\big((a_2 + b_2i) &+ (a_3 + b_3i)\big) = (a_1 + b_1i)\big(a_2 + a_3 + (b_2 + b_3)i\big) \\ &= a_1(a_2 + a_3) - b_1(b_2 + b_3) + \big(a_1(b_2 + b_3) + (a_2 + a_3)b_1\big)i \\ &= a_1a_2 + a_1a_3 - b_1b_2 - b_1b_3 + (a_1b_2 + a_1b_3 + a_2b_1 + a_3b_1)i \\ &= a_1a_2 - b_1b_2 + (a_1b_2 + a_2b_1)i + a_1a_3 - b_1b_3 + (a_1b_3 + a_3b_1)i \\ &= (a_1 + b_1i)(a_2 + b_2i) + (a_1 + b_1i)(a_3 + b_3i). \end{split} \]

• Existência do inverso multiplicativo. Seja \(a + bi \in \bbF_{p^2}\) diferente de \(\overline{0} + \overline{0}i\). Então \(a \neq \overline{0}\) ou \(b \neq \overline{0}\). Se \(a = \overline{0}\) e \(b \neq \overline{0}\), então \(a^2 + b^2 = b^2\). Por \(\bbZ_p\) ser corpo, então, como \(b \neq \overline{0}\), temos que \(b^2 \neq \overline{0}\). Logo, \(a^2 + b^2 \neq \overline{0}\) e, consequentemente, é invertível. De maneira similar, chega-se a esse resultado, supondo que \(a \neq \overline{0}\) e \(b = \overline{0}\). Suponha, agora, que ambos \(a \neq \overline{0}\) e \(b \neq \overline{0}\). Se \(a^2 + b^2 = \overline{0}\), então \(a^2 = -b^2\). Uma vez que \(b \neq \overline{0}\), então \(b^2 \neq \overline{0}\), logo é invertível. Disso segue que \((\frac{a}{b})^2 = -\overline{1}\), ou seja, \(-\overline{1}\) é um quadrado em \(\bbZ_p\). Contudo, \(p \equiv 3 \pmod{4}\), o que significa, conforme o Exercício 1, da Lista 9, que \(-\overline{1}\) não é um quadrado em \(\bbZ_p\). Isso nos dá um absurdo. Logo, \(a^2 + b^2 \neq \overline{0}\) e, por conseguinte, \(a^2 + b^2\) possui inverso em \(\bbZ_p\), \(\frac{1}{a^2 + b^2}\). Portanto, o inverso multiplicativo de \(a + bi\) é \(\frac{a - bi}{a^2 + b^2}\), pois \[ (a + bi)\frac{(a - bi)}{a^2 + b^2} = \frac{a^2 + b^2}{a^2 + b^2} = \overline{1}. \]

Portanto, \(\bbF_{p^2}\) é um corpo.
