Lista 3

Considere a sequência de Fibonacci definida por \(F_0 = F_1 = 1\) e \(F_n = F_{n-1} + F_{n-2}\), para todo \(n \ge 2\) e seja \(\varphi = \frac{1 + \sqrt{5}}{2}\). Mostre que \[ \varphi^{n-1} \le F_n \le \varphi^n \] vale para todo \(n \ge 0\).

Primeiramente, observe que \(\sqrt{5} > 2\), logo \(\varphi^{-1} = \frac{2}{1 + \sqrt{5}} < \frac{2}{3} < 1\). Assim, \(\varphi^{-1} \le F_0 \le \varphi^{0}\), ou seja, a afirmação é verdadeira para \(n = 0\). Vamos mostrar que ela vale para todo \(n \in \bbN\). Começaremos mostrando a segunda desigualdade, isto é, \(F_n \le \varphi^n\). Para isso, mostraremos, antes, que, para todo \(n \in \bbN\), \[ \varphi^{n+1} = F_{n}\varphi + F_{n-1}. \] Faremos isso por indução em \(n\). Assim, se \(n = 1\), então, como \(\varphi\) é raiz da equação \(x^2 - x - 1 = 0\), temos que \(\varphi^2 = \varphi + 1 = F_1\varphi + F_0\). Suponha que \(\varphi^{k+1} = F_{k}\varphi + F_{k-1}\), para \(k > 1\). Então, \(\varphi^{k+2} = \varphi^{k+1}\varphi = F_{k}\varphi^2 + F_{k-1}\varphi\). Usando, novamente que \(\varphi\) é raiz da equação \(x^2 - x - 1 = 0\), isto é, que \(\varphi^2 = \varphi + 1\), temos que \[ \varphi^{k+2} = F_{k}(\varphi + 1) + F_{k-1}\varphi = (F_{k} + F_{k-1})\varphi + F_k = F_{k+1}\varphi + F_k, \] o que prova que \(\varphi^{n+1} = F_{n}\varphi + F_{n-1}\), para todo \(n \in \bbN\). Disso segue que \(\varphi^{n+1} \ge F_{n}\varphi\) e, uma vez que \(\varphi > 1\), obtemos \(\varphi^{n} \ge F_{n}\). Vamos mostrar, agora, que \(\varphi^{n-1} \le F_n\) para todo \(n \in \bbN\) também por indução. Se \(n = 1\), então \(\varphi^0 = 1 \le 1 = F_1\). Suponha que \(\varphi^{k-1} \le F_k\), para \(k > 1\). Então \[ F_{k+1} = F_k + F_{k-1} \overset{\text{HI}}{\ge} \varphi^{k-1} + \varphi^{k-2} = \varphi^{k-2}(\varphi + 1) = \varphi^{k-2}\varphi^2 = \varphi^k, \] ou seja, \(\varphi^k \le F_{k+1}\). Portanto, demonstramos que \(\varphi^{n-1} \le F_n \le \varphi^n\), para todo \(n \ge 0\).
