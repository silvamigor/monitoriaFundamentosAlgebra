Seja \(n \in \bbN\) e ponha \(p_1 = 6n + 1\), \(p_2 = 12n + 1\), \(p_3 = 18n + 1\). Mostre que se \(p_1\), \(p_2\), e \(p_3\) são primos, então \(p_1p_2p_3\) é um número de Carmichael. Usando computador, determine todos os valores de \(n\) entre \(1\) e \(100\) para os quais as condições são satisfeitas.

Vamos usar o Teorema de Korselt, visto na aula Números de Carmichael, para mostrar que \(p_1p_2p_3\) é um número de Carmichael. Para isso precisamos mostrar que \(p_1p_2p_3\) é livre de quadrado, ou seja, que \(p_1p_2p_3\) não é divisível por \(p^2\), para nenhum primo \(p\), e que se \(p\) é um primo que divide \(p_1p_2p_3\), então \(p - 1\) divide \(p_1p_2p_3 - 1\). Observe que a primeira condição segue da hipótese de que \(p_1\), \(p_2\), e \(p_3\) são primos. Isso implica que os únicos primos que dividem \(p_1p_2p_3\) são \(p_1\), \(p_2\) e \(p_3\), ou seja, não há primo cujo quadrado divide \(p_1p_2p_3\). Observe, agora, que \[ p_1p_2p_3 - 1 = 36n(36n^2 + 11n + 1). \] Assim, \(p_1 - 1 = 6n\), \(p_2 - 1 = 12n\) e \(p_3 - 1 = 18n\) dividem \(p_1p_2p_3 - 1\). Com isso, mostramos que \(p_1p_2p_3\) é um número de Carmichael. Vamos determinar, por fim, todos os valores de \(n\) entre \(1\) e \(100\) para os quais \(p_1\), \(p_2\), e \(p_3\) são primos. Para isso, usaremos o Python. Defina primeiro a função eprimo que retorna True, se a entrada for primo, e False, se a entrada for um número composto.

def eprimo(num):
for n in range(2, int(num**1/2) + 1):
if num%n == 0:
return False
return True

Defina as funções p1, p2 e p3 que decrevem, respectivamente, os números \(p_1\), \(p_2\) e \(p_3\).

def p1(num):
return 6*num + 1

def p2(num):
return 12*num + 1

def p3(num):
return 18*num + 1

Para finaliza, utilize os comandos

val = []
for i in range(100):
if eprimo(p1(i + 1)) and eprimo(p2(i + 1)) and eprimo(p3(i + 1)):
val.append(i + 1)

Assim, a lista dos valores val vai armazenar os valores desejados que são: \(1\), \(6\), \(35\), \(45\), \(51\), \(55\), \(56\) e \(100\).
